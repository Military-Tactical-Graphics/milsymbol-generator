The Unit Symbol Generator supports all point symbols from standards listed above. Tactical graphics, other than points are not supported and are disabled in the select menues.

The generator is using milsymbol for rendering the points symbols. This will soon be extended. You can get the complete code for this page, the unit symbol generator extension on GitHub.

This page and milsymbol extension are built and maintained by Marek Szeles and Ondřej Podolský of the Military Tactical Graphics CZ working group. If you want to get in touch please send an email to marek.szeles@eforce.cvut.cz.

The original page and milsymbol is built and maintained by Måns Beckman. If you want to support the development or simply get in touch please send an email to mans.beckman@spatialillusions.com, it is always fun to hear what people are using the symbol generator for.

If you want to use the unit generator in an offline system, send me an email and I will provide you with the necessary files. (The unit generator is pure html/javascript and does not require any server components.)
